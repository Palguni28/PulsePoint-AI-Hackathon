## üöÄ Features

* **AI-Powered Curation**: Automatically finds the most energetic/viral moments in a video using **Gemini 1.5 Flash**.
* **Smart Cropping**: Detects faces using **MediaPipe** and crops standard 16:9 video to 9:16 mobile format, keeping the speaker in focus.
* **Viral Captions**: Adds "Alex Hormozi" style pop-in captions.
* **Deterministic Pipeline**: Logic-first approach (No hallucinations).
